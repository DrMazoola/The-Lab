# Home Assistant

Home Assistant running bare metal on a NUC.  Trying to do as much local as possible and within reason.

## Current Add-ons
* ESPHome Device Builder
* File Editor
* Matter Server
* Mosquitto broker
* Piper
* Samba share
* Speech-to-Phrase
* Terminal & SSH
* Whispter
* Z-Wave JS UI

## Current HACS
* [Pi-hole V6 Integration](https://github.com/bastgau/ha-pi-hole-v6)
* [Pi-hole Card](https://github.com/homeassistant-extras/pi-hole-card)
* [Chime TTS](https://github.com/nimroddolev/chime_tts)
* [SSH](https://github.com/zhbjsh/homeassistant-ssh)
* [Shairport Sync Media Player](https://github.com/parautenbach/hass-shairport-sync)

* [NotiScript](https://github.com/sejnub/ha-notiscript)

My sincere thanks to everyone who is supporting these great features.
